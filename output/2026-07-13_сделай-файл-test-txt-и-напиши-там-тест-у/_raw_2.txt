```python
# main.py

def create_test_file():
    """
    Создает файл test.txt и записывает в него строку "Тест успешный".
    
    Returns:
        None
    
    Raises:
        Exception: Если произошла ошибка при создании файла.
    """
    try:
        with open('test.txt', 'w', encoding='utf-8') as file:
            file.write("Тест успешный")
        print("Файл test.txt создан успешно.")
    except Exception as e:
        print(f"Ошибка при создании файла: {e}")
        raise

if __name__ == "__main__":
    create_test_file()
```

This version of the code includes proper error handling, a docstring, and improved readability. The `encoding='utf-8'` parameter ensures that the file is written with UTF-8 encoding, which supports a wide range of characters including Cyrillic. If an exception occurs during the file creation process, it will be caught, printed, and then re-raised to allow for further handling if needed.
